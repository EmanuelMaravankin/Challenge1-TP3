# Challenge 1 - TP3

Aplicación Android desarrollada con **Jetpack Compose** y **Material 3**.

## Pantallas

| Pantalla | Descripción |
|---|---|
| Welcome | Pantalla de bienvenida con ilustración y botones de acceso |
| Login | Formulario de inicio de sesión con email y contraseña |
| Register | Formulario de registro con confirmación de contraseña |

## Tecnologías

- **Kotlin** 2.2.10
- **Jetpack Compose** (BOM 2024.09.00)
- **Material 3**
- **Navigation Compose** 2.7.7
- **Single Activity** pattern

## Requisitos

- Android Studio Hedgehog o superior
- SDK mínimo: Android 7.0 (API 24)
- SDK objetivo: Android 16 (API 36)

## Cómo correr el proyecto

```bash
# Debug
./gradlew assembleDebug

# Release
./gradlew assembleRelease
```
